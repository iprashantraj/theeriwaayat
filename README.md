# Riwaayat Publications — Website

Mobile-first static website for **Riwaayat Publications**.

> Where Every Voice Becomes a Legacy

## Stack
- Pure HTML / CSS / JavaScript (no build step)
- Google Fonts: Della Respira (display), Harmattan (body)
- Deployed on Vercel: **theeriwaayat.vercel.app**

## Local preview
Just open `index.html` in a browser, or run a tiny local server:

```bash
npx serve .
# or
python3 -m http.server 8000
```

## Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm i -g vercel
vercel              # first deploy (preview)
vercel --prod       # production
```
When prompted for project name, use `theeriwaayat` so the URL becomes `theeriwaayat.vercel.app`.

### Option 2 — GitHub + Vercel dashboard
1. Push this folder to a GitHub repo
2. Go to https://vercel.com/new
3. Import the repo
4. Set the project name to `theeriwaayat`
5. Deploy — that's it

No build settings needed; Vercel will serve the static files as-is.
